### Grade 5: Sleep = Immune Recharge

**Topic:** How sleep affects the immune system  
**Duration:** 45-60 minutes  
**Modeling Skill:** Setting/Modifying Activity Levels  
**Standards:** NGSS 5-LS1-1, 5-PS3-1

[View Lesson](./5th-Grade-Sleep-Immune-Recharge/)

**What Students Learn:**
- Build a digital model with 5 components (Sleep, Immune Strength, White Blood Cells, Germs, Your Energy)
- Connect components to show cause-and-effect relationships
- Run simulations comparing 0% sleep vs 100% sleep
- Analyze graphs and draw evidence-based conclusions about health

**Key Activities:**
1. Building Your Body's Defense Team
2. Connecting the Defense Chain
3. Testing With NO Sleep (0%)
4. Testing With ENOUGH Sleep (100%)
